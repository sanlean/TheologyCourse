# Blasfêmia contra o Espírito Santo

Mateus 12:31-32 é a referência disso.

:::caution Trabalho

Pesquisar a dar opinião pessoal sobre o que é a blasfêmia contra o Espírito Santo (Mateus 12:31-32 e Marcos 3:28-29). Explicar se ela vem do nada e se ela é somente verbal ou é uma progressão, isto é, se são ações ou palavras

:::
